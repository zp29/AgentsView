AGENTSVIEW

连接你的 Agent 星图

输入本机 AgentsView 的访问令牌。令牌只用于换取安全会话，AgentsView 不会将它写入浏览器存储。

会话使用 HttpOnly Cookie；访问令牌不会写入 localStorage 或 URL。
第一阶段功能边界

审查后把产品收敛为“本机 Agent 状态与审批入口”。这能在较短周期内做成，也为后续功能保留清晰接口。

第一阶段保留

  • 三个状态入口与任务下钻
  • 实时状态、当前动作和结果摘要
  • 单次允许与拒绝
  • 中英文与手机适配
  • 本机登录、审计记录和断线提示

延后，避免臃肿

  • 完整终端、聊天与文件浏览器
  • 永久允许规则与批量审批
  • 跨供应商精确费用核算
  • 任意命令输入、暂停与接管
  • 多主机、团队权限与公网部署
接入建议：Codex 使用 app-server 的线程状态与审批请求；Claude Code 优先使用 Agent SDK 的 canUseTool 回调。Hooks 更适合作为通知和兼容层。
